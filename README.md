# Task Tracker CLI

**Консольный менеджер задач на Python**

## Описание проекта

Task Tracker CLI - это простой и удобный консольный менеджер задач, который позволяет управлять вашими задачами прямо из терминала. Проект разработан с использованием современных подходов Python и предоставляет все необходимые функции для эффективного управления задачами.

## Возможности

**Управление задачами** - Создание, обновление и удаление задач
**Статусы задач** - Отслеживание прогресса (todo, in-progress, done)
**Фильтрация задач** - Просмотр задач по статусу
**Хранение данных** - Автоматическое сохранение в JSON формате
**Простой интерфейс** - Интуитивно понятные команды CLI

## Установка

### Требования

- Python 3.13+
- Poetry (для управления зависимостями)

### Установка через Poetry

```bash
# Клонируйте репозиторий
git clone https://github.com/kavlan-dev/task-tracker.git
cd task-tracker

# Установите зависимости
poetry install
```

### Установка без Poetry

```bash
# Клонируйте репозиторий
git clone https://github.com/kavlan-dev/task-tracker.git
cd task-tracker
```

## Использование

После установки вы можете использовать команду `task-cli` для управления задачами.

### Основные команды

```bash
# Показать помощь
task-cli help

# Добавить новую задачу
task-cli add "Купить продукты"

# Посмотреть все задачи
task-cli list

# Посмотреть задачи по статусу
task-cli list todo
task-cli list in-progress
task-cli list done

# Обновить описание задачи
task-cli update 1 "Купить продукты в магазине"

# Удалить задачу
task-cli delete 1

# Пометить задачу как в процессе
task-cli mark-in-progress 1

# Пометить задачу как выполненную
task-cli mark-done 1
```
